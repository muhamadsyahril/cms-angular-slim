#CMS WITH ANGULAR - SLIM FRAMEWORK#


###Requirment###
npm
```
install node js & npm
```
yeoman
```
npm install -g yo
```
grunt
```
npm install -g grunt-cli
```


###Install###
Clone Repository
```
git clone git@github.com:muhamadsyahril/cms-angular-slim.git
```
Migrate database
```
bin/phpmig migrate

```

Create Entity

```
sudo yo angular-slim:entity [entityname]

```
